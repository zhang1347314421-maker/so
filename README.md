# 流光漫剧 (Liuguang ScriptViz) - 单文件版

这是一个集成了所有功能的单文件应用。

## 使用方法

1. 只需要保留 `index.html` 文件。
2. **直接双击** `index.html` 即可在浏览器中运行（推荐使用 Chrome 或 Edge）。
3. 无需安装 Node.js，无需配置环境。

## API 配置

首次打开后，请点击中间栏顶部的 **“⚙️ API 配置”** 按钮，填入您的服务商 Key：
- **剧本分析**: 兼容 OpenAI 格式的 LLM 接口
- **图片**: Comfly / Flux 接口
- **视频**: 云雾 / Luma 接口

数据会自动保存在浏览器的本地缓存中。
